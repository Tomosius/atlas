# Atlas

MCP server that gives AI coding agents structured project knowledge.

## Installation

```bash
uvx atlas-mcp
```

## Usage

Configure in your MCP-compatible editor (Claude Desktop, Zed, etc.):

```json
{
  "mcpServers": {
    "atlas": {
      "command": "uvx",
      "args": ["atlas-mcp"]
    }
  }
}
```
